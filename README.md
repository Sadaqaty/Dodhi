# 🐄 Dodhi: Digital Accounting Suite for Milkmen

**Dodhi** is a premium, Urdu-localized Android application built to modernize the traditional milk management business. It replaces the messy, time-consuming paper "Parchi" registers with a robust, offline-first digital ledger.

## 🧡 The Heart of the Project

This solution was built with a specific purpose: to solve the everyday challenges of **Mamu Atta Muhammad**, a professional milkman (Dodhi) from my home village.

For years, Mamu spent hours every day manually calculating quantities and financial balances from a heavy paper register. When we couldn't find a satisfying digital solution on the market, we decided to build one from scratch.

> [!NOTE]
> **A Collaborative Creation**: Majority of the features in this app—from the "Daily Run" flow to the Urdu PDF reports—were added based on direct requests and consultation with Mamu Atta Muhammad. It is built by a milkman, for milkmen.

---

## ✨ Core Features

- **Digital "Parchi" Register**: Chronological ledger of all milk collections and distributions.
- **Dual-Role Support**: Manage both **Providers** (buying milk) and **Consumers** (selling milk) in one app.
- **Daily Run Workflow**: Record sessions in seconds with a "one-tap" checkmark system.
- **Automatic Financials**: Real-time balance calculation based on customizable rates and quantities.
- **Urdu Localization**: Full support for Urdu, ensuring accessibility for local users.
- **Premium PDF Reports**: Generate professional reports in English or Urdu to share with customers via WhatsApp.
- **Zero-Configuration Backup**: Automatic cloud sync ensures data is never lost, even if the phone is.
